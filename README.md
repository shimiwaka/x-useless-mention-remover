# X Useless Mention Remover

Chrome extension that removes unwanted "via @YouTube" mentions when sharing to X (Twitter).

## How it works

Removes the `via` parameter from X share URLs, preventing mentions like "via @YouTube" from appearing in your posts.

## Installation

1. Open `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select this directory
